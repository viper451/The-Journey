# FOR FRONTEND 

STEPS
1)cd frontend 
2)npm install
3) npm start 


# FOR BACKEND

STEPS
1)cd backend
2)npm install 
3)node server.js

BOTH THE SERVER AND CLIENT WILL START RUNNING AT PORT 1337 AND 5000  Respectively
